FEATURES

Here are Some of the Capabilites of Our App

We are developing a Disc Golf game in which we have a character throw a disc into different targets to gain points. The medium will be Oculus Rift using the remotes. It will be fully virtual so it will not interact with the real environment. It runs by using velocity of the remote to control the speed and the angle at which the frisbee is thrown controls the direction at which the disc flies and dictate the spin. The player can grab the discs without having to move.
